# Greeting Skill

A simple example skill that demonstrates how to create agent skills.

## When to Use

Use this skill when the user asks for a greeting or wants to say hello.

## Instructions

When activated, provide a friendly, personalized greeting. Consider:

1. Time of day (morning, afternoon, evening)
2. Any context from the conversation
3. Keep it brief and warm

## Example Responses

- "Good morning! How can I help you today?"
- "Hello there! Ready to assist with your coding tasks."
- "Hi! What are we working on?"
